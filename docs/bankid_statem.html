Description

BankID State Machine.

A simple state machine that implements the BankID flow. Do not use this in a production system. This module is only useful for very simple cases like testing and development.

 {ok,Pid} = bankid_statem:start_link({self(), ClientOpts}).
 {ok,_Order} = bankid_statem:auth(Pid, IpAddr, PersonalNumber, [{}]).
 receive
   {bankid, {complete, _CompletionData}} -> ok;
   {bankid, {failed, _Failure}} -> error
 end.
