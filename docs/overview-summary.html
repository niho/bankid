This is a client implementation for the Swedish BankID Relaying Party API (v5.1). The client can be used and tested with the official BankID test environment. To use it with the production environment you will need to sign an agreement with a Swedish bank and request SSL certificates that will give access to the production environment.

For a list of banks that offer BankID as a service, please see the official guides for companies and public sector.
